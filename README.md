## 🎯 What is FADE?

FADE is not just another productivity app. It's a statement. A philosophy. A way of life.

Designed with the modern minimalist in mind, FADE combines stunning glassmorphism design with powerful productivity features that actually work. No bloat. No BS. Just pure, elegant functionality.

### Why FADE?

- **Glassmorphism UI** - Frosted glass effects that look illegal
- **Focus Timer** - Pomodoro on steroids with customizable intervals
- **Smart Tasks** - AI-powered task prioritization
- **Daily Reflections** - Journal your wins and losses
- **Statistics** - Beautiful charts that make you motivated
- **Dark Mode First** - Because we're not savages

---

## 📱 Features

### 🔥 Focus Mode
Deep work sessions with customizable timers (25/50/90 minutes). The UI literally fades away to minimize distractions. Get in the zone.

### ✅ Smart Tasks
- Priority-based task management
- Drag and drop reordering
- Quick add with keyboard shortcuts
- Task categories and tags
- Completion tracking

### 📊 Statistics
Beautiful, motivating statistics that show:
- Total focus hours
- Tasks completed
- Current streak
- Weekly progress

### 📝 Daily Journal
Reflect on your day. Track your progress. Build self-awareness.

### 🎨 Design Philosophy
- **Glassmorphism** - Translucent, frosted-glass aesthetics
- **Minimalism** - Every pixel has a purpose
- **Smooth Animations** - 60 FPS or nothing
- **Color Psychology** - Carefully chosen gradients for motivation

## 🚀 Getting Started

### Prerequisites
- Xcode 15.0+
- iOS 17.0+
- macOS Sonoma 14.0+
- Swift 5.9+

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/gedeonkoh/FADE-app.git
cd FADE-app
```

2. **Open in Xcode**
```bash
open FADE.xcodeproj
```

3. **Build and Run**
- Select your target device or simulator
- Press `Cmd + R` to build and run
- Experience productivity nirvana
